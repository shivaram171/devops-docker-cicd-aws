# devops-docker-cicd-aws


DevOps CI/CD Pipeline with Docker and AWS

This project demonstrates a simple DevOps pipeline.

Tools Used
- Git
- GitHub
- Jenkins
- Docker
- AWS EC2
- Ubuntu

Pipeline Flow

Developer pushes code to GitHub
↓
Jenkins triggers CI pipeline
↓
Docker builds container image
↓
Image deployed to AWS EC2 server
↓
Application runs inside Docker container
